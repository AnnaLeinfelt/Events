## Mat & dryck

### Catering

- Gör egna lappar för att märka ut allergier, det är inte säkert att det kommer med.
- Beställ alltid 5 extar till fikat.
- Kolla upphandling och kontakat dem ca 4-5 veckor innan eventet
- Ange fika tid till 15-30 minuter inna den faktiska fika tiden för att säkerställa att fikat kommer i tid.
